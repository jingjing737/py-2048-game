# py-2048-game 🎮

终端版 2048 游戏，使用 Unicode + 颜色渲染。

## 快速开始

```bash
python3 2048.py
```

## 操作

| 按键 | 功能 |
|------|------|
| `↑↓←→` | 移动方块 |
| `R` | 重新开始 |
| `Q` | 退出 |

## 特性

- 彩色方块（不同数字不同颜色）
- 合并动画 + 得分统计
- 达到 2048 提示胜利
- 无路可走自动结束
- 无第三方依赖（仅用标准库）
