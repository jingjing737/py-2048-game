# py-2048-game 🎮

Terminal-based 2048 game built with Python and print rendering.

## Quick Start

```bash
python3 2048.py
```

## Controls

- `↑` `↓` `←` `→` — Move tiles
- `W` `A` `S` `D` — Alternative controls
- `R` — Restart
- `Q` — Quit

## Features

- Pure ASCII rendering — no curses, works in any terminal
- Fixed-width grid layout — no alignment issues
- Score tracking
- Win/lose detection
